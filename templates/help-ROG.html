Help Center

Frequently Asked Questions

How do I upload student documents?
To upload student documents, navigate to the Upload Documents page and follow these steps:
  1. Enter the student's Application ID
  2. Select the transcript file (PDF format)
  3. Select the application data file (PDF format)
  4. Click the Upload button
How do I review applications?
To review applications:
  1. Go to the Review Applications page
  2. Select an application from the list
  3. Review the student's information and documents
  4. Make your decision and add any comments
How do I generate reports?
To generate reports:
  1. Navigate to the Generate Reports page
  2. Select the type of report you need
  3. Choose the date range if applicable
  4. Click Generate Report
  5. Download the report in your preferred format
